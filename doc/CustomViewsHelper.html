- (String) sort_orders(objs)

This function is used when rendering sort selections to the new/edit Custom View flows. Renders a string like "7:asc,13:desc,4:asc", which is interpreted by the Javascript on the page.

def sort_orders(objs)
  orders = ""

  orders = objs.map {|obj| "#{obj.display_field_id}:#{obj.sort_direction}" }.join(',') unless objs.blank?
end
