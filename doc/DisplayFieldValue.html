Overview

A DisplayFieldValue represents the intersection between a SurveyResponse and a DisplayField. It is a single data point within a SurveyResponse and can represent a respondent's answer to a question or admin-defined data.

def self.search(criteria = {})
  search_scope = nil
  criteria.each do |k, criteria_fields|
    search_scope = search_scope | generate_scope(criteria_fields) if criteria_fields.delete('clause_join') == 'OR'
    search_scope = search_scope & generate_scope(criteria_fields) if criteria_fields.delete('clause_join') == 'AND'
  end
end
